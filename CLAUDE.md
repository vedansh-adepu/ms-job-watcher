# ms-job-watcher

Automated job board alerter: polls 6 hardcoded company APIs + ~1,200 ATS boards (Greenhouse, Lever, SmartRecruiters, Workday, Ashby) every 20–30 min via GitHub Actions and emails new US software-engineering roles.

---

**HARD RULE: At the start of every session, read `docs/STATE.md` before doing anything. After ANY meaningful change, bug found/fixed, or decision, update `docs/STATE.md` before finishing — never leave it stale.**
